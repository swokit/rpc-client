# Swokit

Swokit - Swoole kit library. 

Some useful components for use swoole.

Components:

- `conn-pool`
- `server`
- `rpc-server`
- `http-server`
- `websocket-server`
- `task`
- `utils`
- `process`

## Install

```bash
composer require swokit/swokit
```

## Git Subtree

- add git repo:

```bash
./subtree-add.sh NAME
```

- pull git repo:

```bash
./subtree-pull.sh NAME|all
```

- push to repos:

```bash
./subtree-push.sh NAME|all
```
